# What is the project about?

**smTraker.com** (short: **smT.com**) is a social media tracker that monitors user channels, provides key statistics, and displays the latest uploaded video. Its goal is to deliver clear and easy-to-understand insights about what’s happening on a user’s or another creator’s channel.

The project is currently in **beta**, so some bugs may occur. The full release is planned for **mid to late summer 2026**.
